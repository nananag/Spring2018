Quality: About This Data Set

The dataset is in high quality overall. But there are a few missing values in the dataset.

Is the data complete?

The data misses a few logs on the first and last day of data collection period. It is obvious to see based on the total duration of both status being much smaller than the rest of the days.

Is the data coherent?

The data makes sense.

Is the data correct?

The duration of screen being unlocked and locked should add up to a same number across the data collection period. I briefly checked the sum, and it matches.
